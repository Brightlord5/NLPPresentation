Text Classification

Text classification involves categorizing text into organized groups. Common applications include spam detection and sentiment analysis.

Text Classification Code


import pandas as pd
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB


# Sample data
data = {
    'text': ['I love this!', 'This is bad', 'I am happy', 'I am sad'],
    'label': ['positive', 'negative', 'positive', 'negative']
}
df = pd.DataFrame(data)


# Vectorize the text
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(df['text'])


# Train a simple Naive Bayes classifier
model = MultinomialNB()
model.fit(X, df['label'])
# Predict
predictions = model.predict(X)
print(predictions)
